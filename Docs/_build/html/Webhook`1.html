Creates a new instance of the [Webhook1](Webhook1#webhook1) object for a webhook which has already been registered with Trello.

Parameter: id

The id.

Parameter: auth

(Optional) Custom authorization parameters. When not provided, TrelloAuthorization.Default will be used.
